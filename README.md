# Infotech International

**Slogan:** Technology Beyond Boundaries

Infotech International is an original educational ecommerce platform inspired by common large-marketplace patterns. It does not copy Amazon copyrighted design, branding, layout, or assets.

## Module 1 Included

- Professional project structure for frontend, backend, database, assets, and docs.
- Responsive homepage with header, navigation, hero, categories, featured products, flash sale, best sellers, recommendations, brands, reviews, stats, news, videos, newsletter, loading screen, dark mode, toasts, skeleton loading, live search, voice-search support, wishlist/cart counters, and compare state.
- Original logo, favicon, and product artwork.
- Express server entry point with a no-install preview fallback.
- MySQL schema foundation for future modules.

## How To Run

1. Install Node.js.
2. Open this folder in a terminal.
3. Run npm install.
4. Run npm start.
5. Open http://localhost:3000.

If dependencies are not installed, src/server.js falls back to a tiny built-in preview server. Install dependencies before continuing to authentication, security, and database modules.
